# PNS-PS7-19-20-pns-ps7-19-20-ihm

### Setting workspace

1. Clone repository and open a terminal in *polympic-capacitor*
2. run ```yarn global add vue-native-cli```
3. run ```yarn global add ionic```
4. run ```yarn install```
5. run ```yarn run build```

### Using application
